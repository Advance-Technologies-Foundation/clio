## Remote Module Reference Guides

**MANDATORY:** Read and follow these guides before working on any task related to Creatio Freedom UI integrations with remote modules.

### Styling & theming (read from this project)

These guides ship with the project under `./ai-guides/`. Whenever a component has visuals (any `.scss` or styled template, colors, typography, spacing, or theming), reading them is mandatory:

- Styling a runtime component (Shadow DOM, design-token consumption): ./ai-guides/REMOTE_COMPONENT_STYLING_AI_GUIDE.md
- Design token catalog (all `--crt-*` token names and default values): ./ai-guides/DESIGN_TOKENS_AI_GUIDE.md
- Making a custom element look like a built-in Creatio element (per-element recipes): ./ai-guides/REMOTE_COMPONENT_NATIVE_LOOK_AI_GUIDE.md
