# manage-windows-features

Install windows features required for Creatio.

## Usage

```bash
clio manage-windows-features [options]
```

## Description

Install windows features required for Creatio.

## Aliases

`mng-win-features`, `mwf`

## Examples

```bash
clio manage-windows-features [options]
```

## Options

```bash
-c, --Check
    Check required feature states
-i, --Install
    Install required features
-u, --Uninstall
    Uninstall required features
```
