# configure-workspace

Legacy heading for the `cfg-worspace` command.

## Synopsis

```bash
clio cfg-worspace --Packages <PKG1,PKG2> [OPTIONS]
```

## Description

The source command is named `cfg-worspace`, while the GitHub command reference uses the
more descriptive heading `configure-workspace`.

## Examples

```bash
clio cfg-worspace --Packages PkgOne,PkgTwo -e dev
```
