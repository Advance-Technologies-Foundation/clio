# uninstall-app-remote

Uninstall an application from a Creatio environment.

## Synopsis

```bash
clio uninstall-app-remote <APP_NAME> [OPTIONS]
```

## Description

Removes an installed application package from the selected environment.

## Examples

```bash
clio uninstall-app-remote MyApp -e dev
```
