# restore-nuget-pkg

Restore a NuGet package to a local folder.

## Synopsis

```bash
clio restore-nuget-pkg [OPTIONS]
```

## Description

Downloads or restores the requested NuGet package contents into a local folder.

## Examples

```bash
clio restore-nuget-pkg --help
```
