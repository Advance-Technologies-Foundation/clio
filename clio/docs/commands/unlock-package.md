# unlock-package

Unlock a package in Creatio.

## Usage

```bash
clio unlock-package [<Name>] [options]
```

## Description

Unlock a package in Creatio.

## Aliases

`up`

## Examples

```bash
clio unlock-package [<Name>] [options]
```

## Arguments

```bash
Name
    Package name
```
