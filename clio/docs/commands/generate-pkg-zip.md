# generate-pkg-zip

Prepare an archive of creatio package.

## Usage

```bash
clio generate-pkg-zip [<Name>] [options]
```

## Description

Prepare an archive of creatio package.

## Aliases

`comp-pkg`, `compress`

## Examples

```bash
clio generate-pkg-zip [<Name>] [options]
```

## Arguments

```bash
Name
    Name of the compressed package
```

## Options

```bash
-d, --DestinationPath <VALUE>
    Full destination path for gz file
-p, --Packages <VALUE>
-s, --SkipPdb
```
