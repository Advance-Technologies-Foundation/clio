# set-app-icon

Set application icon.


## Usage

```bash
clio set-app-icon [options]
```

## Description

Set application icon.

## Aliases

`ai`, `appicon`, `set-icon`

## Examples

```bash
clio set-app-icon [options]
```

## Options

```bash
-p, --app-name <VALUE>
App name
-i, --app-icon <VALUE>
Application icon path. Required.
-f, --app-path <VALUE>
Path to application package folder or archive. Required.
```
