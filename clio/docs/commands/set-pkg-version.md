# set-pkg-version

## Command Type

    Service commands

## Name

set-pkg-version - set package version

## Description

Set specified package version into descriptor.json by specified package path.

## Example

```bash
clio set-pkg-version <PACKAGE PATH> -v <PACKAGE VERSION>
```
