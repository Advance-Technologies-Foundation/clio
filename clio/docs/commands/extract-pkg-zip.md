# extract-pkg-zip

Extract a packaged application or package archive.

## Usage

```bash
clio extract-pkg-zip [<Name>] [options]
```

## Description

Extract a packaged application or package archive.

## Aliases

`extract`, `unzip`

## Examples

```bash
clio extract-pkg-zip [<Name>] [options]
```

## Arguments

```bash
Name
    Name of the compressed package
```

## Options

```bash
-d, --DestinationPath <VALUE>
    Destination path for package folder
```
