# install-nuget-pkg

Install a NuGet package into a Creatio environment.

## Synopsis

```bash
clio install-nuget-pkg [OPTIONS]
```

## Description

Installs a NuGet package into the selected web application or Creatio environment.

## Examples

```bash
clio install-nuget-pkg --help
```
