# merge-workspaces

Merge multiple workspaces.

## Synopsis

```bash
clio merge-workspaces [OPTIONS]
```

## Description

Combines multiple workspaces into a single resulting workspace layout.

## Examples

```bash
clio merge-workspaces --help
```
