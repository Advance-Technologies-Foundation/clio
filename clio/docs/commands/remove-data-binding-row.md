# remove-data-binding-row

Remove a row from a package data binding.

## Usage

```bash
clio remove-data-binding-row [options]
```

## Description

Remove a row from a package data binding.

## Examples

```bash
clio remove-data-binding-row [options]
```

## Options

```bash
--package <VALUE>
    Target package name. Required.
--binding-name <VALUE>
    Binding folder name. Required.
--key-value <VALUE>
    Primary-key value of the row to delete. Required.
--workspace-path <VALUE>
    Workspace root path. Defaults to the current workspace
```
