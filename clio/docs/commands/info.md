# info

Show clio, cliogate, and .NET runtime versions.

## Usage

```bash
clio info [options]
```

## Description

Show clio, cliogate, and .NET runtime versions.

## Aliases

`get-version`, `i`, `ver`

## Examples

```bash
clio info [options]
```

## Options

```bash
--all
    Get versions for all known components
-s, --settings-file
    Get path to settings file
--clio
    Get clio version
--gate
    Get clio-gate version
--runtime
    Get dotnet version
```
