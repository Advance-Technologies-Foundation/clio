# push-nuget-pkg

Publish a NuGet package to a feed.

## Synopsis

```bash
clio push-nuget-pkg [OPTIONS]
```

## Description

Use `push-nuget-pkg` to upload a prepared NuGet package to a configured NuGet server.

## Examples

```bash
clio push-nuget-pkg --help
```
