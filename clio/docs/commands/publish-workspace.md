# publish-workspace

Alias for `publish-app`.

## Synopsis

```bash
clio publish-workspace <WORKSPACE_PATH> [OPTIONS]
```

## Description

`publish-workspace` is implemented as an alias of `publish-app` and publishes a workspace
either to a zip file or to an application hub folder.
