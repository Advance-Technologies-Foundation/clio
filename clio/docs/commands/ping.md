# ping

Alias for `ping-app`.

## Synopsis

```bash
clio ping <ENVIRONMENT_NAME> [OPTIONS]
```

## Description

`ping` verifies connectivity and credentials for the selected environment.
It is an alias of the canonical `ping-app` command documented in the repository.
