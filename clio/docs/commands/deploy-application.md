# deploy-application

Deploy an application from one environment to another.

## Synopsis

```bash
clio deploy-application <APP_NAME> [OPTIONS]
```

## Description

`deploy-application` transfers an application package from a source Creatio environment
to a destination environment.

## Examples

```bash
clio deploy-application MyApp -e source -d target
```
