# clone-env

Clone a registered environment definition.

## Synopsis

```bash
clio clone-env <SOURCE_ENVIRONMENT> <TARGET_ENVIRONMENT>
```

## Description

Creates a copy of an existing local environment definition in clio settings.

## Examples

```bash
clio clone-env dev test
```
