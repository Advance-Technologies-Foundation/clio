# add-schema

Create a schema file in a workspace package.

## Usage

```bash
clio add-schema <SchemaName> [options]
```

## Description

Create a schema file in a workspace package.

## Examples

```bash
clio add-schema <SchemaName> [options]
```

## Arguments

```bash
SchemaName
    Schema Name. Required.
```

## Options

```bash
-p, --package <VALUE>
    Package path or name. Required.
-t, --type <VALUE>
    Schema type. Required.
```
