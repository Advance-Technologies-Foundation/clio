# delete-skill

Delete a managed skill.

## Usage

```bash
clio delete-skill [options]
```

## Description

Delete a managed skill.

## Examples

```bash
clio delete-skill [options]
```

## Options

```bash
--skill <VALUE>
    Managed skill name to delete. Required.
--scope <VALUE>
    Skill target scope: workspace or user. Defaults to workspace
```
