# pack-nuget-pkg

Pack a NuGet package.

## Synopsis

```bash
clio pack-nuget-pkg [OPTIONS]
```

## Description

Creates a NuGet package from the configured input sources and package metadata.

## Examples

```bash
clio pack-nuget-pkg --help
```
