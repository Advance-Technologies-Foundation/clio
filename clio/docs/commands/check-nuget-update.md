# check-nuget-update

Check whether updated NuGet packages are available.

## Synopsis

```bash
clio check-nuget-update [OPTIONS]
```

## Description

Use this command to check for newer versions of supported Creatio-related NuGet packages.

## Examples

```bash
clio check-nuget-update
```
