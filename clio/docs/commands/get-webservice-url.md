# get-webservice-url

Get configured web service URLs.

## Synopsis

```bash
clio get-webservice-url [OPTIONS]
```

## Description

Displays the currently configured base URL values for web service entries in the target environment.

## Examples

```bash
clio get-webservice-url -e dev
```
