NAME
    publish-workspace - alias for publish-app

DESCRIPTION
    The publish-workspace command is implemented as an alias of publish-app.
    It publishes a workspace either to a zip file or to an application hub folder.

SYNOPSIS
    clio publish-workspace <WORKSPACE_PATH> [OPTIONS]

OPTIONS
    <WORKSPACE_PATH>
        Path to the workspace folder to publish

    -f, --file <ZIP_PATH>
        Publish workspace to a zip file

    -r, --repo-path <WORKSPACE_PATH>
        Explicit workspace folder path

    -h, --app-hub <FOLDER_PATH>
        Application hub output folder when not publishing to a file

    -a, --app-name <APP_NAME>
        Application name when publishing to an application hub folder

    -v, --app-version <VERSION>
        Application version to write into publish output

    -b, --branch <BRANCH>
        Branch name used for app hub publishing

EXAMPLES
    clio publish-workspace ./workspace -f ./MyApp.zip
        Publish a workspace to a zip file

    clio publish-workspace ./workspace --app-hub ./hub --app-name MyApp --app-version 1.0.0
        Publish a workspace to an application hub folder
