NAME
    info - Show clio, cliogate, and .NET runtime versions

USAGE
    clio info [options]

DESCRIPTION
    Show clio, cliogate, and .NET runtime versions.

ALIASES
    get-version, i, ver

EXAMPLES
    clio info [options]

OPTIONS
    --all
        Get versions for all known components
    -s, --settings-file
        Get path to settings file
    --clio
        Get clio version
    --gate
        Get clio-gate version
    --runtime
        Get dotnet version
