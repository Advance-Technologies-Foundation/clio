NAME
    set-application-icon - alias heading for set-app-icon

DESCRIPTION
    The Commands.md reference uses this alias heading for the set-app-icon command.
    Use it to update the icon for an application package.

SYNOPSIS
    clio set-app-icon [OPTIONS]

OPTIONS
    Supports the same options as the canonical set-app-icon command.

EXAMPLES
