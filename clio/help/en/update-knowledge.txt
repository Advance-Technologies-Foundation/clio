COMMAND TYPE
    Integrations & tools

NAME
    update-knowledge - Update installed Clio knowledge to the latest verified version

SYNOPSIS
    clio update-knowledge

DESCRIPTION
    Downloads and verifies a strictly newer stable package in staging, then
    atomically changes the activation marker. The previous version remains as
    a last-known-good fallback. An already-running MCP process observes the new
    marker on its next knowledge lookup; no MCP restart is required.

EXAMPLES
    clio update-knowledge
    clio info-knowledge

EXIT CODES
    0   Updated, or already up to date
    1   Not installed, unavailable, rejected, or publishing failed
