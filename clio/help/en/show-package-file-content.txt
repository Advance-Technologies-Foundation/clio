NAME
    show-package-file-content - Show package file content

DESCRIPTION
    Displays the content of a file stored within a package.

REQUIREMENTS
    This command requires the cliogate package to be installed on the target
    Creatio environment. If cliogate is not installed, the command displays an
    error and exits. Install or update it with:
        clio install-gate -e <ENVIRONMENT_NAME>

SYNOPSIS
    clio show-package-file-content [OPTIONS]

OPTIONS
    Supports the canonical show-package-file-content command options.

EXAMPLES
    clio show-package-file-content --help
        Display canonical options and usage examples

SEE ALSO
    pull-pkg - Download package files locally
    install-gate - Install or update the cliogate package (required)
