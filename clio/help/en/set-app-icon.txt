NAME
    set-app-icon - Set application icon

USAGE
    clio set-app-icon [options]

DESCRIPTION
    Set application icon.

ALIASES
    ai, appicon, set-icon

EXAMPLES
    clio set-app-icon [options]

OPTIONS
    -p, --app-name <VALUE>
        App name
    -i, --app-icon <VALUE>
        Application icon path. Required.
    -f, --app-path <VALUE>
        Path to application package folder or archive. Required.
