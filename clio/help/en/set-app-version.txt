NAME
    set-app-version - Set application version

USAGE
    clio set-app-version [<workspace>] [options]

DESCRIPTION
    Set application version.

ALIASES
    appversion

EXAMPLES
    clio set-app-version [<workspace>] [options]

ARGUMENTS
    workspace
        Workspace folder path

OPTIONS
    -v, --app-version <VALUE>
        Application version. Required.
    -p, --package-name <VALUE>
        Package name
    -f, --package-folder <VALUE>
        Package folder path
