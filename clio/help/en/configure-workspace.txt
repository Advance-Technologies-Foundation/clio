NAME
    configure-workspace - legacy heading for cfg-worspace

DESCRIPTION
    The Commands.md reference describes workspace configuration under this heading,
    while the implementation uses the cfg-worspace command name.

SYNOPSIS
    clio cfg-worspace --Packages <PKG1,PKG2> [OPTIONS]

OPTIONS
    --Packages <PKG1,PKG2>
        Comma-separated package list to ensure in the workspace

    -e, --Environment <ENVIRONMENT_NAME>
        Environment name to save into workspace metadata

EXAMPLES
    clio cfg-worspace --Packages PkgOne,PkgTwo -e dev
        Configure workspace packages and bind the workspace to the dev environment
