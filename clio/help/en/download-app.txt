NAME
    download-app - alias for download-application

DESCRIPTION
    The download-app command is an alias for download-application.
    It exports an application package from the selected Creatio environment to a local file.

SYNOPSIS
    clio download-app <APP_NAME> [OPTIONS]

OPTIONS
    <APP_NAME>
        Application name to download

    -e, --Environment <ENVIRONMENT_NAME>
        Source environment name

    -f, --FilePath <FILE_PATH>
        Target file path for the exported application package

    -u, --uri
    -l, --Login
    -p, --Password
        Direct connection parameters instead of using -e

EXAMPLES
    clio download-app MyApplication -e dev
        Download an application from the dev environment

    clio download-app MyApplication -e dev -f ./MyApplication.gz
        Download to a specific output file
