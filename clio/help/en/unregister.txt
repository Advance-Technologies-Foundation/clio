NAME
    unregister - Remove clio shell integrations

USAGE
    clio unregister [options]

DESCRIPTION
    Remove clio shell integrations.

EXAMPLES
    clio unregister [options]

OPTIONS
    -t, --Target <VALUE>
        Target environment location. Could be user location or machine location. Use
        'u' for set user location and 'm' to set machine location. Default: u.
    -p, --Path <VALUE>
        Path where clio is stored
