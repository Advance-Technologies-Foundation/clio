NAME
    show-web-app-list - List registered Creatio environments

USAGE
    clio show-web-app-list [<App name>] [options]

DESCRIPTION
    List registered Creatio environments.

ALIASES
    env, envs, show-web-app

EXAMPLES
    clio show-web-app-list [<App name>] [options]

ARGUMENTS
    App name
        Name of application

OPTIONS
    -e, --env <VALUE>
        Environment name (alias for positional)
    -s, --short
        Show short list
    -f, --format <VALUE>
        Output format: json, table, raw. Default: json. Default: json.
    --raw
        Raw output (no formatting) - shorthand for --format raw
