NAME
    set-pkg-version - Set package version

USAGE
    clio set-pkg-version <PackagePath> [options]

DESCRIPTION
    Set package version.

ALIASES
    spv

EXAMPLES
    clio set-pkg-version <PackagePath> [options]

ARGUMENTS
    PackagePath
        Package path. Required.

OPTIONS
    -v, --PackageVersion <VALUE>
        Package version. Required.
