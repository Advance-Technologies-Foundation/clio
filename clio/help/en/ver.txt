NAME
    ver - alias for info

DESCRIPTION
    The ver command is an alias for info.
    It displays version information for clio, bundled cliogate, the active .NET runtime,
    and optionally the settings file path.

SYNOPSIS
    clio ver [OPTIONS]

OPTIONS
    --all
        Display all available version information

    --clio
        Display only clio version

    --gate
        Display only bundled cliogate version

    --runtime
        Display only .NET runtime version

    -s, --settings-file
        Display the path to the clio settings file

EXAMPLES
    clio ver
        Display all version information

    clio ver --runtime
        Display only the .NET runtime version

    clio ver -s
        Display the settings file path
