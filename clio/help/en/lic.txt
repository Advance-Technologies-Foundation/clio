NAME
    lic - alias for upload-licenses

DESCRIPTION
    The lic command is a short alias for upload-licenses.
    Use it to upload one or more license files to a target environment.

SYNOPSIS
    clio lic <LICENSE_FILE> [OPTIONS]

OPTIONS
    <LICENSE_FILE>
        Path to a license file to upload

    -e, --Environment <ENVIRONMENT_NAME>
        Target environment name

    -u, --uri
    -l, --Login
    -p, --Password
        Direct connection parameters instead of using -e

EXAMPLES
    clio lic ./license.lic -e dev
        Upload a license file to the dev environment
