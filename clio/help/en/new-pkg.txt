NAME
    new-pkg - Create a new package project

USAGE
    clio new-pkg <Name> [options]

DESCRIPTION
    Create a new package project.

ALIASES
    init

EXAMPLES
    clio new-pkg <Name> [options]

ARGUMENTS
    Name
        Name of the created instance. Required.

OPTIONS
    -r, --References <VALUE>
        Set references to local bin assemblies for development
